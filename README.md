# StudySync — Class 10 Board Exam Companion

A frontend-only student dashboard built with HTML, CSS, and JavaScript.

## Files

- `index.html` — website structure
- `style.css` — Night Desk OS styling and responsive design
- `script.js` — local functionality and Local Storage persistence

## How to open

Double-click `index.html`, or run a local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Notes

- No backend is included.
- Login/signup and dashboard data are stored in the browser using Local Storage.
